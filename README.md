# <img src="images/icon_48.png" width="45" align="left"> Admin Tools V2

Admin Tools V2 is a Chrome extension designed to enhance your experience on Canvas LMS by providing various administrative tools.
## Features
 - **Add Blueprint Parent**
 - **Add Divs to Question Bank**
 - **Admin Account Name**
 - **Associate Sections to Blueprint**
 - **Blueprint Associations**
 - **Breadcrumbs**
 - **Copy all Broken Links**
 - **Cross List**
 - **Delete all Quizzes**
 - **Delete all Quiz Questions**
 - **Editor**
 - **Edit Question Banks**
 - **Get Users Emails**
 - **Lock**
 - **Nav to Module**
 - **Select all Files**
 - **Start End Date Search**

## New Features

- **Ability to Fix Divs for Question Bank Question**:This is to make it easier for a user to fix all of the divs of all of the questions in a question bank. It also works on Pages where you have to click a "More Questions" button
- **New Link Validator Tool**: This is for the link Validator tool for fixing broken links, added features include retrieving the Course Name, A Possible reason for why the link is broken, The Location of the Broken link, The Title of the Link, The Page Url, and The Broken link

## Installation

To install the latest version of Canvas Admin Tools V2, follow these steps:

1. **Download the Extension**:
    - Navigate to the [GitHub repository](https://github.com/vidaldl/Admin-Tools-v2).
    - Find the release corresponding to the latest update.
    - Download the `.zip` file for the release.

2. **Extract the ZIP File**:
    - Once downloaded, locate the `.zip` file on your computer.
    - Extract the contents of the `.zip` file to a folder on your computer. Ensure you remember this location, as you'll need to access this folder to load the extension into Chrome.

3. **Load the Extension in Chrome**:
    - Open the Chrome browser.
    - Access the Extensions page by navigating to `chrome://extensions/` in your browser's address bar or by opening Chrome's menu, selecting "More Tools", and then "Extensions".
    - Enable "Developer mode" by toggling the switch in the top-right corner of the Extensions page.
    - Click the "Load unpacked" button, which becomes visible in developer mode.
    - Navigate to the folder where you extracted the `.zip` file and select it. Chrome will then load the extension.

4. **Verify Installation**:
    - Ensure the extension is listed on the Extensions page and is enabled.
    - You should see the Canvas Admin Tools V2 icon in your Chrome toolbar, indicating the extension is active.

You're now ready to use Canvas Admin Tools V2 with its latest features and improvements!

## Usage

 ### Ability to Fix Divs for Question Bank Question
**How to use:**

1. Navigate to Manage Question Banks on Canvas.
2. Pick a quiz to manage.
3. Right-click the Screen and Press the "Fix Divs for all Questions" Button.
4. The extension will automatically add div elements to each quiz question, making it easier to apply custom styles and manage questions.

### CopyAllBrokenLinks

This feature scans the current page for broken links and copies them to your clipboard for easy review and action.

**How to use:**

1. Navigate to any page on Canvas LMS where you want to check for broken links.
2. Select the `Copy All Broken Links` feature from the dropdown menu.
3. The extension will scan the page for broken links and copy them to your clipboard. You can then paste the list of broken links into a document or email for further action.
